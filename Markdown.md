# Инструкция для работы с Markdown

## Выделение текста
Чтобы выделить текст курсивом, необходимо обрамить зведочками *. Например, *вот так*

Чтобы выделить текст полужирным, необходимо обрабить его звездочками **. Например, **вот так**
